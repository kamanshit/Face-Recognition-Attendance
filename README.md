👨‍🏫 Face Recognition Attendance System
A complete face recognition–based attendance system built using Python, OpenCV, and Streamlit. It captures face data, trains a recognition model, and marks attendance in real-time using webcam input.

🎯 Features
✅ Face detection and recognition using OpenCV (LBPH model)

🎥 Live webcam-based recognition

🕒 Automatic attendance logging with date and time

📊 Streamlit UI for live video + attendance display

🗃️ CSV file for exported attendance records

🔧 Modular code: data collection, model training, recognition UI

🛠️ Tech Stack
Python

OpenCV

Streamlit

Pandas, NumPy

Pickle

Pickle

📝 Notes
Make sure your camera is accessible and lighting is sufficient

You can add more people later by re-running collect_faces.py and train_model.py

Supports real-time multi-face recognition
